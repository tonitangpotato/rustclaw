## Hindsight vs Engram 对比分析

### 背景
- potato 发了 https://github.com/vectorize-io/hindsight 让我对比
- 这个分析之前 session 做过但没存，导致重复劳动

### Hindsight 核心
- SaaS 级 agent memory（PostgreSQL + Docker + server/client）
- 4 层记忆：World Facts, Experiences, Observations, Mental Models
- TEMPR 4 路检索：Semantic + BM25 + Graph + Temporal，RRF 融合
- 关键能力：Observation Consolidation（自动从 facts 合成高阶 insight）、reflect()（LLM 在记忆上推理）、Entity Graph
- LongMemEval benchmark 91.4%

### Engram 优势
- 嵌入式 Rust crate，单 SQLite，零依赖
- ACT-R 衰减 + Hebbian 关联（真认知科学基础）
- 本地隐私，90ms 查询
- 7 种记忆类型（含 emotional, procedural, opinion）

### Engram 差距
1. 没有 Observation Consolidation（最大差距）
2. 没有 Entity Graph
3. 没有多路检索融合（只有 FTS5）
4. 没有时间推理
5. 没有 reflect() 推理能力

### 可借鉴
1. Observation Consolidation — consolidate() 应该调 LLM 合成新知识
2. Entity Extraction + Graph — retain 时提取 entities/relationships
3. 多路 Recall 融合 — 加 embedding 向量搜索 + entity graph + RRF
