## Engram Hub: Seed Strategy + Data Model Decision
- **Seed Strategy**: 不等用户贡献，从 Reddit/SO/CSDN/HF/GitHub Issues 抓取讨论 → LLM 提取经验 → 打包为 engram domain packages
- 本质：semantic-level indexed search → 分领域 SQLite search（ACT-R + Hebbian）
- Seed 成本 <$50，10 个领域 × 1000+ 记忆
- **数据模型决策**：不用完整 KG，在 Hebbian links 上加 link_type (causes/solves/contradicts/supersedes) + confidence + source
- 80% KG 能力，20% 复杂度
- potato 提出关键问题："engram 还是 KG？还是 causal links？还是已有知识做KB新知识做engram？"
- 结论：统一模型，typed Hebbian links 是最优解
- **Link 准确性问题**：LLM 提取会有 correlation≠causation 错误 + 领域模糊性
- **三层防线**：① 提取时 LLM 自评 confidence（明确0.9/隐含0.6/模糊0.3降级co_occurrence）② 使用时 confidence 做 recall 权重 ③ 社区修正用后反馈
- **关键保证**：低 confidence typed link worst case = 普通 Hebbian，不比现状差
- 实现优先级：confidence 字段 P0 → 保守默认 co_occurrence P0 → 阈值 0.7 才展开因果链 P0 → verified 字段 P1
- 已更新 engram-hub-discussion.md (§10 Seed Strategy, §12 Data Model, §12 Link Accuracy) + IDEAS.md IDEA-20260406-03
