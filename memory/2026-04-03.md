## Intake: ChromaFs — Virtual Filesystem for AI Assistants
- Source: https://x.com/densumesh/status/2039765361533637016 (@densumesh, Mintlify)
- 用虚拟文件系统替代 sandbox micro-VM 给 AI agent 提供文档浏览能力
- Session 创建 46s→100ms，850K conversations/month 零边际成本
- 基于 just-bash (Vercel Labs TS bash 实现) + Chroma 向量库
- 关键技巧：gzipped path tree、smart grep (Chroma coarse → Redis prefetch → in-memory fine)、EROFS 只读
- 与我们的 agent 工具设计相关：agentic filesystem pattern 是 RAG 的替代方案
- Saved: intake/twitter/densumesh-2039765361533637016.md
